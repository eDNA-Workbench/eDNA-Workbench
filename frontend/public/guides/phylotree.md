# Phylogenetic Viewer
A tool for visualizing phylogenetic trees.

**Input Required:**
*   **Tree File:** A **Newick** format file, with or without branch values (e.g., `COI_MLtree.nwk`).

**Functionalities:**
*   **Visualization:** View the tree structure.
*   **Node Management:** Tick to show internal node names; click internal nodes to collapse/reduce subtrees; rename nodes.
*   **Resizing:** Resize the view horizontally or vertically.
